# hybrid-identity-project

Hybrid Identity Lab: Entra ID and Active Directory

This project demonstrates the implementation of hybrid identity and access management using Microsoft Entra ID and Active Directory. It includes user provisioning, group-based access control, and a ticket-based challenge where a Finance department and contractor access were added to simulate a real-world scenario.

Key Skills:

Microsoft Entra ID (Azure AD)
Active Directory (AD DS)
Role-Based Access Control (RBAC)
Identity and Access Management (IAM)
User provisioning and group management
